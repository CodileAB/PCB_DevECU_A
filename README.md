Wolley DevECU A
===============

This is the KiCad project for the PCB for the Wolley development ECU, DevECU A.

It has an Infineon XMC1404 MCU with buffered and ESD protected GPIO.

Specifications
--------------

| Name              | Value |
| ----------------- | ----- |
| PCB layers        | 2     |
| Control Interface | CAN   |

Key Components
--------------

| Component | Part | Details |
| ----------| ---- | --------|
| MCU | Infineon XMC1404_F064X200A | ARM Cortex-M0, 32-bit, 48 MHz, 200 kB flash |
| CAN transceiver | Infineon TLE7251VSJ | |
